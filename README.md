# DirectLab 🏥

DirectLab is a full-stack diagnostic lab booking platform that allows patients to book lab tests directly, schedule home sample collection, track booking status, and chat with lab technicians.

## 🚀 Features

- 🔐 Authentication with role-based access (Patient / Lab)
- 🏥 Lab registration & approval system
- 📅 Test booking with date & time selection
- 💬 Real-time chat between patients and labs
- 📊 Booking status tracking
- 🔒 Secure database with Row Level Security (RLS)

## 🛠 Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (Auth + PostgreSQL + Storage)
